◆ nextBlock()

Block TrackModel.Switch.nextBlock ( Block  currBlock)

Returns the nextBlock when calling a switch.

If the switch is true, and is called on a root, it returns the lower leaf block. If called on the lower leaf, it returns the root. If called on the higher leaf, returns the higher leaf. If false and is called on root, returns the higher numbered block. If called on the lower leaf, returns the lower leaf. If called on the higher block returns root.
