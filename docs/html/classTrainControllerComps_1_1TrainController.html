Detailed Description

This class is a GUI that is used to control a selected train.

The selected train is a Train object that is selected by using the train drop down menu.

This class communicates with other components such as:

TCTrainInfoPanel - displays basic train info such as speed, power, authority, and suggested speed. TCSpeedController - allows the user to control the train's speed.

Andrew Lendacky
