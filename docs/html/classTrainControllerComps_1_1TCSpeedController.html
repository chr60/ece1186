Detailed Description

This class is responsible for allowing the user to set the speed of the selected train if in Manual mode, or controls the speed automatically if in Automatic mode.

This class collaborates with the Train and Train Controller class.

Regulates the train's speed using Power Law.

Bug:
Sometimes the train's speeds oscillates between 0.0 and 0.8 when stopped.

This happens sometimes when stopping at a station, or at an authority. If the train's speed is greater than the set speed, the train's service brake is pressed to slow the train down.

This is called every second.

◆ setSetSpeed()

void TrainControllerComps.TCSpeedController.setSetSpeed ( Double  setSpeed)

Sets the set speed that the train should go.

In manual mode, if the set speed if greater than the suggested speed, the speed of the train is set to the block speed.

In Automatic mode, if the set speed is greater than the suggested speed, the speed of the train is set to the suggested speed.
