Detailed Description

This class is responsible for display the states of the 3 failures that can occur on the train, as well as sending a signal to the CTC to repair the broken systems.

The three failures are: antenna, power, and brake.

This class collaborates with the Train Controller and Train class.

void TrainControllerComps.TCFailures.refreshUI ( )

This method is not called from anywhere else, so the window does not refresh itself when a train has been fixed.

Therefore, it will continue to show the old states of the failures until the user closes the window and reopens it. Refreshes the UI to update the radio buttons based on the status of the train.
