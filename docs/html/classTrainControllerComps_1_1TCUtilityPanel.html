Detailed Description

A component of the Train Controller responsible for displaying, and transmitting information and changes to the train corresponding to the Air Conditioning unit (AC), Heating unit, Left/Right Doors, and Lights.

This class collaborates with a Train Model, and the TrainController class, which it is a sub-component of.

Author
Andrew Lendacky

TrainControllerComps.TCUtilityPanel.TCUtilityPanel ( )

Constructor for creating a new TCUtiltiyPanel object where the initial positions of the radio button are in the 'OFF' position, and there is no selected train.

By default, this constructor sets the 'inManualMode' to false. Selected train must be set by the Train Controller before being used.
